# Lost & Found Management System

A Flask + NLP based web application to manage lost and found items on campus.  
Users can add new items, search for items, and view all reports.  
The system uses text similarity and Sentence Transformers for intelligent matching.

## Installation
1. Clone this repository:
   git clone https://github.com/YourUsername/lost-found-system.git
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   flask run

## Usage
- Open http://127.0.0.1:5000/ in your browser.
- Use the options:
  - Add a new item
  - Search for an item
  - Show all items
